## About

### 나의 일상을 그려내는 그림일기 ##긋다

어린 시절 그림 일기장처럼, <br />
나 자신에게 조금이나마 집중하는 시간을 갖길 바라는 마음에서 서비스를 기획하게 되었습니다.

나 자신에게 마음을 열어주세요. <br />
그림을 그리며 나의 하루에 말을 걸어보세요. <br />
오늘은 너무 평범한 날인 동시에 미래를 잇는 가장 소중한 시간이니까요! <br />
그러니 나의 일상들에 무심하지 말고 더 특별하게 대해주세요.

소중한 일상을 기억해내는 또 다른 방법, "긋다" <br />
내 생각과 감정들이 오롯이 새겨진 일기장이 될 거에요.

이제 나만의 갤러리를 채워나가보세요!
